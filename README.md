# TIME FRAME

駅伝・長距離ランナー専用の画像ジェネレーター「TIME FRAME」

大会・記録会や練習後に、ランナーが自分の記録を"かっこよく・誇らしく"SNSで共有できる画像を簡単に生成できるWebツールです。

## 機能

- ✅ テキスト入力 → 画像生成 → ダウンロード
- ✅ Instagram対応（1080×1080 PNG）
- ✅ PC / スマホ両対応（レスポンシブ）
- ✅ サーバー保存なし（プライバシー重視）
- ✅ 無料・登録不要
- ✅ PWA対応（オフライン利用可能）
- ✅ リアルタイムバリデーション
- ✅ アクセシビリティ対応（ARIA、キーボードナビゲーション）
- ✅ キーボードショートカット対応
- ✅ トースト通知システム

## 使い方

1. 大会名・種目・記録を入力
2. 背景画像をアップロード
3. プレビューを確認
4. ダウンロードボタンをクリック
5. SNSでシェア！

## 技術スタック

- **フレームワーク**: React 19 + TypeScript
- **ビルドツール**: Vite 6
- **スタイリング**: Tailwind CSS 4
- **画像生成**: Canvas API
- **デプロイ**: Cloudflare Pages対応

## 開発

```bash
# 依存関係のインストール
npm install

# 開発サーバーの起動
npm run dev

# ビルド
npm run build

# プレビュー（ビルド後）
npm run preview
```

## デプロイ

Cloudflare Pagesへのデプロイ:

1. GitHubリポジトリに push
2. Cloudflare Pages でリポジトリを接続
3. ビルドコマンド: `npm run build`
4. 出力ディレクトリ: `dist`
